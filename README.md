Rest api를 이용한 롤 전적 검색

server폴더
- summoner.json : 소환사 정보 -> 3001포트에 서버돌리기
- history.json : 소환사 대전기록 정보 -> 3002포트에 서버돌리기

명령어
json-server --watch ./summoner.json --port 3001
json-server --watch ./history.json --port 3002
